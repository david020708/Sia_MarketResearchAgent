**报告日期**: 2026年3月

## 1. GTM模式总览

中国低空经济的GTM（Go-to-Market）模式因细分赛道不同而差异显著。当前阶段，政府是最大的"客户"和"渠道"，政策本身就是最强的分销力量。

| GTM模式 | 适用赛道 | 典型企业 | 占比 |
|---------|---------|---------|------|
| **政府采购/招标** | 低空基建、公共安全、应急 | 莱斯信息、四川九洲 | ~40% |
| **B2B直销** | 工业无人机、电力巡检 | 大疆企业版、纵横股份 | ~25% |
| **平台模式** | 无人机物流、低空出行 | 美团、亿航 | ~10% |
| **经销商/代理** | 消费级无人机、农业植保 | 大疆经销网络 | ~15% |
| **运营服务** | 低空旅游、培训、巡检外包 | 中信海直、各地运营商 | ~10% |

## 2. 五大GTM路径详解

### 2.1 路径一：政府采购驱动（当前主流）

**模式描述**: 地方政府通过产业基金、专项债、财政拨款采购低空基础设施和服务。

**渠道结构**:
```
中央政策 → 省级规划 → 市级低空经济专班 → 政府采购招标 → 企业中标
```

**关键数据**:
- 33个省市竞逐低空经济（[36氪](https://m.36kr.com/p/2915789685922693)）
- 2024年15省市设立产业基金超1,170亿元（[圆象低空产业观察](https://yuanxiangsky.com/N/131.html)）
- 十五五规划将低空经济列为国家战略（[圆象低空产业观察](https://yuanxiangsky.com/N/889.html)）

**GTM策略要点**:
- 建立地方政府关系是第一优先级
- 参与地方低空经济规划编制，抢占标准话语权
- 通过示范项目建立标杆案例
- 央企/国企背景在政府采购中具有天然优势

**适用企业**: 低空基建（UTM、起降场、通信导航）、公共安全无人机

### 2.2 路径二：B2B行业直销

**模式描述**: 针对特定行业客户（电力、农业、建筑等）的直接销售和解决方案交付。

**渠道结构**:
```
行业需求 → 解决方案设计 → 直销团队/行业BD → POC验证 → 规模化部署
```

**GTM策略要点**:
- 深耕垂直行业，建立行业know-how壁垒
- 从"卖硬件"转向"卖解决方案+服务"
- 与行业头部客户共建标杆案例
- 大疆在工业级市场以绝对优势领跑（[中国报告网](https://m.chinabaogao.com/detail/753766.html)）

**适用企业**: 工业无人机、电力巡检、农业植保

### 2.3 路径三：平台模式

**模式描述**: 构建低空服务平台，连接供给端（飞行器、飞手）和需求端（消费者、企业）。

**渠道结构**:
```
消费者/企业需求 → 平台APP → 调度系统 → 飞行器运营 → 服务交付
```

**关键案例**:
- **美团无人机配送**: 获CAAC全国全域物流运营许可，"一张牌照，全国通飞"模式（[Low Altitude Economy Aero](https://lowaltitudeeconomy.aero/evtol-news-and-electric-aircraft-news/cargo-drones/meituan-fourth-generation-drone-caac-national-full-territory-logistics-license)）
- **亿航低空出行**: 探索类似网约车的空中出行平台

**GTM策略要点**:
- 先在试点城市跑通单位经济模型
- 利用全国性牌照快速复制
- 平台网络效应是核心壁垒
- "本地深耕，全球扩展"的新范式（[Daily Economic](https://cn.dailyeconomic.com/en/2025/05/29/rooted-locally-expanding-globally-a-new-low-altitude-economy-business-paradigm-driven-by-technolog/)）

**适用企业**: 无人机物流、低空出行、低空旅游

### 2.4 路径四：经销商/代理网络

**模式描述**: 通过多层级经销商网络覆盖终端用户。

**渠道结构**:
```
品牌商 → 全国总代 → 区域代理 → 零售门店/电商 → 终端用户
```

**关键案例**:
- **大疆**: 全球最成熟的无人机经销网络，线上（官网、天猫、京东）+ 线下（DJI Store、授权经销商）
- **极飞科技**: 农业无人机通过农资经销商渠道下沉

**GTM策略要点**:
- 消费级产品依赖品牌力和渠道覆盖
- 农业产品需要与农资渠道深度绑定
- 售后服务网络是关键竞争力

**适用企业**: 消费级无人机、农业无人机

### 2.5 路径五：运营服务模式

**模式描述**: 不卖设备，卖服务。以飞行小时、作业面积、数据量等为计费单位。

**渠道结构**:
```
客户需求 → 运营服务商 → 飞行作业 → 数据交付/服务完成 → 按量计费
```

**GTM策略要点**:
- 轻资产模式，降低客户采购门槛
- 区域化运营，建立本地服务能力
- 从单一服务向综合低空服务平台演进

**适用企业**: 低空旅游运营、巡检服务、培训机构

## 3. 合作生态图谱

中国低空经济的合作生态正在快速形成：

### 3.1 核心生态参与者

| 角色 | 代表企业/机构 | 生态位 |
|------|-------------|--------|
| 飞行器制造商 | 大疆、亿航、峰飞、小鹏汇天 | 硬件供给 |
| 平台运营商 | 美团、顺丰、京东 | 需求聚合 |
| 通信基建 | 华为、中兴、中国移动/联通/电信 | 5G低空网络 |
| 空管系统 | 莱斯信息、中电科 | UTM/空域管理 |
| 地方政府 | 深圳、合肥、成都等 | 政策+基建+采购 |
| 保险 | Swiss Re等（[Swiss Re](https://www.swissre.com/institute/research/topics-and-risk-dialogues/china/insuring-low-altitude-airspace-economy-china.html)） | 风险保障 |
| 咨询/研究 | KPMG（[KPMG报告](https://assets.kpmg.com/content/dam/kpmgsites/cn/pdf/zh/2025/10/the-golden-age-of-low-altitude-economy.pdf)）、赛迪 | 行业洞察 |
| 学术机构 | 清华、北航、南航 | 技术研发 |
| 资本 | VC/PE、地方产业基金 | 资金供给 |

### 3.2 电信运营商的角色

STL Partners的研究指出，电信运营商和基础设施提供商正在形成合作生态，支撑低空运营（[STL Partners](https://stlpartners.com/research/lesson-from-chinas-low-altitude-economy/)）：
- 提供5G低空通信连接
- 参与UTM系统建设
- 低空数据服务

### 3.3 ZTE（中兴通讯）的布局

中兴通讯已发布低空作业价值场景端到端解决方案及商业投资分析（[ZTE](https://www.zte.com.cn/content/zte-site/www-zte-com-cn/china/about/magazine/zte-technologies/2026/1/3/10.html)），从通信设备商向低空经济解决方案提供商延伸。

## 4. 渠道经济学

### 4.1 政府采购渠道

| 指标 | 数据 |
|------|------|
| 平均项目周期 | 6-18个月 |
| 平均项目金额 | 500万-5亿元 |
| 毛利率 | 30-50% |
| 回款周期 | 3-12个月 |
| 关键成功因素 | 政府关系、资质、标杆案例 |

### 4.2 B2B直销渠道

| 指标 | 数据 |
|------|------|
| 平均销售周期 | 3-6个月 |
| 客单价 | 10万-500万元 |
| 毛利率 | 40-60% |
| 复购率 | 高（年度服务合同） |
| 关键成功因素 | 行业know-how、产品性能、售后 |

| 指标 | 数据 |
|------|------|
| 平均客单价 | 2,000-15,000元 |
| 毛利率 | 50-65%（大疆） |
| 渠道分成 | 线上10-15%，线下20-30% |
| 关键成功因素 | 品牌、产品力、渠道覆盖 |

## 5. GTM趋势判断

### 短期（2026-2027）
- 政府采购仍是最大收入来源
- 美团等平台模式开始规模化验证
- "本地深耕"是所有企业的必经之路

### 中期（2028-2030）
- 平台模式占比将显著提升
- B2B行业解决方案走向标准化
- 出海成为头部企业的第二增长曲线

### 长期（2030+）
- 低空出行平台可能成为最大的GTM渠道（类似滴滴模式）
- 数据服务和增值服务成为新的利润中心
- 中国标准输出带动企业全球化

## 6. VC投资视角：GTM能力评估框架

评估低空经济创业公司时，建议关注以下GTM能力维度：

| 维度 | 权重 | 评估要点 |
|------|------|---------|
| 政府关系 | 30% | 是否有地方政府合作、示范项目 |
| 行业深度 | 25% | 是否深耕特定垂直行业 |
| 技术壁垒 | 20% | 产品是否有不可替代的技术优势 |
| 规模化能力 | 15% | 商业模式是否可复制 |
| 团队背景 | 10% | 是否有航空+互联网复合背景 |
