## 1. 范围定义

| 维度 | 定义 |
|------|------|
| **市场** | 中国低空经济（eVTOL、无人机、低空基础设施、低空运营服务全产业链） |
| **地理范围** | 中国大陆 |

## 2. 政治环境（Political） — 评分: 5/5（极度有利）

### 2.1 国家战略定位

中国低空经济享有罕见的政治支持力度：

- **连续三年写入政府工作报告**（2024、2025、2026年），从"积极打造"升级为"新兴支柱产业"（[中国eVTOL新闻](https://www.chinaevtolnews.com/p/china-elevates-aerospace-and-low)）
- **十五五规划纲要五次点名低空经济**，从国家战略到项目清单的顶层设计跃迁（[圆象低空产业观察](https://yuanxiangsky.com/N/889.html)）
- **国家发改委**明确表态"低空经济发展前景可期"（[国家发改委](https://www.ndrc.gov.cn/wsdwhfz/202411/t20241129_1394729.html)）
- **十部门联合发布低空经济发展指南**，跨部委协调力度空前（[腾讯新闻](https://news.qq.com/rain/a/20260205A023TW00)）

### 2.2 地方政府竞争

- **33个省市竞逐低空经济**，形成"城市争夺战"格局（[36氪](https://m.36kr.com/p/2915789685922693)）
- 深圳、合肥、成都为三大重点城市（[OurChinaStory](https://www.ourchinastory.com/zh/15209/)）
- 2024年全国15省市设立低空产业基金，总金额超1,170亿元（[圆象低空产业观察](https://yuanxiangsky.com/N/131.html)）
- 四川设立30亿元低空经济专项基金（[东方财富](https://finance.eastmoney.com/a/202505133403228097.html)）
- 杭州组建30亿低空基金（[圆象低空产业观察](https://yuanxiangsky.com/N/139.html)）

### 2.3 政治风险评估

| 风险因素 | 评估 |
|---------|------|
| 政策连续性 | 极高 — 已写入五年规划，不会因领导层变动而逆转 |
| 地缘政治风险 | 中等 — 无人机技术属于敏感领域，大疆已受美国制裁，但国内市场不受影响 |
| 产业政策转向风险 | 极低 — 当前处于政策加速期 |
| 地方保护主义 | 中等 — 各地竞相设立产业基金可能导致重复建设 |

### 2.4 两会最新动态（2026年）

- 2026年两会提出四项低空经济立法提案（[Low Altitude Economy Aero](https://lowaltitudeeconomy.aero/evtol-news-and-electric-aircraft-news/low-altitude-economy/china-two-sessions-2026-lae-legislative-proposals)）
- 低空经济立法取得新进展（[同花顺](http://news.10jqka.com.cn/20260314/c675297326.shtml)）

**政治环境评分: 5/5** — 低空经济是当前中国政策支持力度最强的新兴产业之一，堪比2015-2020年的新能源汽车赛道。

## 3. 经济环境（Economic） — 评分: 4/5（有利）

### 3.1 宏观经济背景

- 2024年中国GDP约126.06万亿元，增速约5%（[新华网](http://www.news.cn/fortune/20251222/1a9ded8c452846a5a858f12866e39f07/c.html)）
- 经济处于结构转型期，政府积极寻找新增长引擎
- 低空经济被视为继新能源汽车后的下一个万亿级赛道

### 3.2 资本市场热度

| 指标 | 数据 | 来源 |
|------|------|------|
| 2025年融资事件增速 | 同比+71% | [虎嗅](https://www.huxiu.com/article/4832461.html) |
| 企业注册量增速 | 同比+141% | [泰伯网](https://www.taibo.cn/p/99088) |
| 相关企业总数 | 超16万家 | [泰伯网](https://www.taibo.cn/p/99088) |
| 产业基金规模 | 超1,170亿元（2024年15省市） | [圆象低空产业观察](https://yuanxiangsky.com/N/131.html) |
| 热钱蓄势 | 超700亿元 | [36氪](https://m.36kr.com/p/2915789685922693) |

### 3.3 产业经济特征

- **政府投资驱动明显**: 当前阶段以基础设施建设和政府采购为主要收入来源
- **商业化尚在早期**: eVTOL等核心应用场景的市场化收入有限
- **人才供需失衡**: 无人机操控员持证27万+，缺口约100万（[千龙网](https://china.qianlong.com/2025/1027/8572802.shtml)）

### 3.4 经济风险

| 风险 | 影响 |
|------|------|
| 地方债务压力 | 可能影响产业基金实际到位率 |
| 经济增速放缓 | 可能延缓商业化场景的消费端需求 |
| 泡沫风险 | 企业注册量暴增141%，部分可能为"蹭概念" |
| 汇率波动 | 对进口核心零部件（芯片、传感器）成本有影响 |

**经济环境评分: 4/5** — 资本热度高、政府投入大，但商业化收入仍需验证，存在泡沫风险。

## 4. 监管环境（Regulatory） — 评分: 4/5（有利，快速完善中）

### 4.1 核心监管机构

| 机构 | 职责 |
|------|------|
| 中国民航局（CAAC） | 适航认证、航空器注册、飞行标准、空域管理 |
| 空军/军方 | 低空空域使用审批（中国空域由军方管控） |
| 工信部（MIIT） | 无人机制造标准、频谱管理 |
| 国家发改委（NDRC） | 产业政策、投资审批 |
| 公安部 | 无人机实名登记、飞行安全 |
| 地方政府 | 低空航线审批、起降场规划、地方性法规 |

### 4.2 关键法规里程碑

| 时间 | 法规/事件 | 意义 | 来源 |
|------|----------|------|------|
| 2023年6月 | 《无人驾驶航空器飞行管理暂行条例》 | 首部无人机专项行政法规 | 国务院 |
| 2024年1月 | 条例正式实施 | 无人机分类管理制度落地 | 国务院 |
| 2025年4月 | 无人机运营合格证纳入市场准入负面清单 | 规范准入门槛 | [圆象低空产业观察](https://yuanxiangsky.com/N/293.html) |
| 2025年12月 | CAAC发布eVTOL适航标准（AC-21-AA-2025-XX） | 统一eVTOL认证框架 | [Low Altitude Economy Aero](https://lowaltitudeeconomy.aero/evtol-news-and-electric-aircraft-news/low-altitude-economy/ac-21-aa-2025-xx-why-chinas-unified-evtol-framework-outpaces-faa-and-easa) |
| 2025年12月 | 《民用航空法》修正案通过 | 无人机首次纳入民航法 | [VisaHQ](https://www.visahq.com/news/2025-12-27/cn/china-passes-landmark-civil-aviation-law-amendment-regulating-drones/) |
| 2025年12月 | 低空经济标准体系建设指南发布 | 313项标准框架 | [新华网](https://www.xinhuanet.com/tech/20260203/6cfae2e482294bcd83281337da6a5077/c.html) |

### 4.3 适航认证

- 中国eVTOL适航认证周期约31个月，远快于西方的7年+（[Low Altitude Economy Aero](https://lowaltitudeeconomy.aero/evtol-news-and-electric-aircraft-news/low-altitude-economy/china-evtol-certification-31-months-vs-7-years)）
- 亿航EH216-S为全球首款获得TC证书的载人eVTOL
- 313项标准框架可能重塑全球eVTOL市场准入规则（[Low Altitude Economy Aero](https://lowaltitudeeconomy.aero/evtol-news-and-electric-aircraft-news/low-altitude-economy/china-lae-standards-313-framework-evtol-certification-market-access)）

### 4.4 企业所需核心资质

根据金杜律师事务所梳理（[金杜](https://www.kwm.com/cn/zh/insights/latest-thinking/low-altitude-economy-overview-of-key-qualification-licenses.html)）：

| 资质 | 适用对象 |
|------|---------|
| 型号合格证（TC） | 航空器制造商 |
| 生产许可证（PC） | 航空器制造商 |
| 适航证（AC） | 每架航空器 |
| 通用航空经营许可证 | 运营企业 |
| 无人机运营合格证（OC） | 无人机运营企业 |
| 操控员执照 | 飞行员/操控员 |

### 4.5 外资准入

- 无人机运营已纳入2025年版市场准入负面清单（[证券日报](http://www.zqrb.cn/finance/hongguanjingji/2025-04-24/A1745458707441.html)）
- 航空器制造领域对外资有一定限制，但鼓励外资参与零部件供应链
- [REASONED INFERENCE] 低空经济涉及空域安全，核心运营环节预计将长期限制外资控股

**监管环境评分: 4/5** — 监管框架快速完善，认证速度全球领先，但空域管制仍由军方主导，审批流程存在不确定性。

## 5. 进入壁垒（Barriers to Entry） — 评分: 3/5（中等壁垒）

### 5.1 壁垒矩阵

| 壁垒类型 | 高度 | 说明 |
|---------|------|------|
| **资本壁垒** | 高 | eVTOL研发需数十亿元投入；基础设施建设资本密集 |
| **技术壁垒** | 高 | 适航认证技术门槛高，电池、飞控、自动驾驶等核心技术要求高 |
| **监管壁垒** | 中-高 | TC/PC/AC认证周期长，空域审批复杂 |
| **渠道壁垒** | 中 | 政府采购关系、地方政府合作资源 |
| **品牌壁垒** | 低-中 | 市场早期，品牌尚未固化 |
| **规模壁垒** | 中 | 大疆在无人机领域已形成规模优势 |
| **人才壁垒** | 中 | 复合型人才（航空+AI+电气）稀缺 |

### 5.2 细分赛道壁垒差异

| 细分赛道 | 进入壁垒 | 关键壁垒 |
|---------|---------|---------|
| eVTOL整机制造 | 极高 | 适航认证、资本、技术 |
| 消费级无人机 | 高 | 大疆垄断地位、供应链规模 |
| 工业无人机 | 中 | 行业know-how、客户关系 |
| 无人机物流 | 中-高 | 空域审批、运营资质、网络效应 |
| 低空基建（硬件） | 中 | 资本、政府关系 |
| 低空基建（软件/UTM） | 中-低 | 技术能力、标准参与 |
| 低空运营服务 | 低-中 | 运营资质、人才 |
| 培训 | 低 | 资质认证 |

### 5.3 国企/央企优势

- 低空经济涉及空域安全和基础设施，国企/央企在政府采购和基建项目中具有天然优势
- 中航工业、航天科技等央企已布局低空经济
- [REASONED INFERENCE] 核心基础设施（空域管理系统、通信导航）预计将以国企为主导

**进入壁垒评分: 3/5** — 壁垒因细分赛道差异大。eVTOL和消费无人机壁垒极高，但低空运营服务和软件领域对创业公司相对友好。

## 6. 市场环境记分卡

| 维度 | 评分（1-5） | 权重 | 加权得分 |
|------|-----------|------|---------|
| 政治环境 | 5 | 30% | 1.50 |
| 经济环境 | 4 | 25% | 1.00 |
| 监管环境 | 4 | 25% | 1.00 |
| 进入壁垒 | 3 | 20% | 0.60 |
| **综合得分** | | **100%** | **4.10 / 5.00** |

**综合评级: 有利（Favorable）**

### Top 3 机遇

1. **政策红利窗口期** — 十五五规划+万亿产业基金，2025-2030年是政策支持最密集的阶段
2. **监管先发优势** — 中国eVTOL认证速度全球领先，313标准框架可能成为国际标准输出
3. **基建投资拉动** — 低空基础设施建设处于从0到1阶段，确定性高、体量大

### Top 3 风险

1. **空域开放不及预期** — 军方主导的空域管制改革进度是最大不确定性
2. **泡沫与过度投资** — 企业注册量暴增141%、地方基金扎堆，可能导致产能过剩和资源浪费
3. **商业化验证滞后** — 政府投资驱动的增长能否转化为可持续的市场化收入尚待验证

## 7. 置信度评估

| 维度 | 置信度 | 说明 |
|------|--------|------|
| 政治环境 | 高 | 政策信号明确、公开可查 |
| 经济环境 | 中-高 | 宏观数据可靠，但产业基金实际到位率不透明 |
| 监管环境 | 中 | 法规框架清晰，但执行层面（尤其空域审批）实操经验有限 |
| 进入壁垒 | 中 | 细分赛道差异大，部分推算基于行业类比 |

**关键假设（若错误将显著改变结论）**:
- 假设空域管制改革将按十五五规划推进 — 若军方态度保守，整个赛道增速将大幅放缓
- 假设地方产业基金能够实际落地 — 若地方债务压力导致资金缩水，基建进度将受影响
- 假设eVTOL商业化在2027-2028年开始规模化 — 若推迟，TAM中最具想象力的部分将延后
