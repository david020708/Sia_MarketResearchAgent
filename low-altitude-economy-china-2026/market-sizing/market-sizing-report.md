## 2. 方法论概述

本报告采用三种独立方法进行市场规模测算，并进行三角验证：

- **方法A — 自上而下（Top-Down）**: 从宏观经济数据和权威机构预测出发，逐层筛选
- **方法B — 自下而上（Bottom-Up）**: 从细分市场和企业级数据汇总
- **方法C — 供给侧（Supply-Side）**: 从已知市场参与者的营收推算

## 4. 市场规模测算

### 方法A — 自上而下（Top-Down）

**逻辑链**: 中国GDP → 战略性新兴产业占比 → 低空经济在新兴产业中的份额

**Step 1 — 宏观基数**
- 2024年中国GDP约126.06万亿元（[国家统计局 via 新华网](http://www.news.cn/fortune/20251222/1a9ded8c452846a5a858f12866e39f07/c.html)）
- 战略性新兴产业增加值占GDP比重约13%（国务院目标为2025年达17%以上）

**Step 2 — 低空经济在战略性新兴产业中的定位**
- 低空经济被列为"新兴支柱产业"，与新能源汽车、半导体并列（[中国eVTOL新闻](https://www.chinaevtolnews.com/p/china-elevates-aerospace-and-low)）
- 赛迪研究院等权威机构测算2024年低空经济规模约5,000亿元（[赛迪 via 人民网](http://finance.people.com.cn/n1/2025/1219/c1004-40627884.html)）

**Step 3 — 验证**
- 5,000亿元 / 126.06万亿元 = 约0.4% GDP，作为一个新兴产业处于早期阶段，该比例合理
- 对比：新能源汽车产业2024年规模约5万亿元（约4% GDP），低空经济约为其1/10，符合发展阶段差异

**Step 4 — 2030年预测**
- 多家权威机构一致预测2030年突破2万亿元（[中国网财经](http://finance.china.com.cn/industry/20251124/6279610.shtml)）
- 隐含CAGR（2024→2030）：(20,000/5,000)^(1/6) - 1 ≈ 26%
- 注意：部分机构给出的CAGR为11.51%（[报告大厅](https://www.chinabgao.com/info/1255248.html)），口径差异可能源于对"低空经济"定义的宽窄不同

**Top-Down 结果**:
| 年份 | 市场规模 |
|------|---------|
| 2024 | ~5,000亿元 |
| 2025 | ~1.5万亿元 |
| 2030 | ~2万亿元 |

> ⚠️ **注意**: 2024→2025年从5,000亿跳至1.5万亿（3倍增长）存在口径差异的可能。5,000亿可能为核心产业口径（整机制造+运营服务），1.5万亿为广义口径（含上下游关联产业、基础设施投资等）。本报告后续将区分核心市场与广义市场。

### 方法B — 自下而上（Bottom-Up）

从各细分市场独立测算后汇总：

#### B1. 民用无人机（消费级+工业级）
- 2025年市场规模：1,200-1,691亿元（[报告大厅](https://m.chinabgao.com/info/1296408.html)；[搜狐](https://www.sohu.com/a/885747591_121924584)）
- 取中值：~1,450亿元
- 其中消费级约占40%（~580亿），工业级约占60%（~870亿）

#### B2. 无人机物流与配送
- 2026年预计突破580亿元（[远瞻慧库](https://www.baogaobox.com/insights/260109000024696.html)）
- 2025年估算：~400亿元（回推，假设年增长率40-50%）

#### B3. eVTOL / 城市空中交通（UAM）
- 亿航2025年营收5.095亿元（[亿航IR](https://ir.ehang.com/news-releases/news-release-details/ehang-reports-fourth-quarter-and-fiscal-year-2025-unaudited/)）
- eVTOL整体市场仍处于极早期，2025年中国eVTOL市场估算：~50-100亿元（含研发投入、试运营收入、政府采购）
- [PROXY — REASONED ESTIMATE] 基于亿航为头部企业占约5-10%市场份额推算

#### B4. 低空基础设施（起降场、空域管理、通信导航监视）
- 2024年全国15省市设立低空产业基金超1,170亿元（[圆象低空产业观察](https://yuanxiangsky.com/N/131.html)）
- 33个省市竞逐，超700亿热钱蓄势（[36氪](https://m.36kr.com/p/2915789685922693)）
- 基础设施投资为当前低空经济最大支出项，2025年估算：~3,000-5,000亿元
- [PROXY — REASONED ESTIMATE] 基于产业基金规模和地方政府基建投入推算

#### B5. 农业植保无人机
- 中国农业植保无人机保有量全球第一
- 2025年市场规模估算：~300-500亿元（含设备+服务）
- [PROXY — REASONED ESTIMATE] 基于大疆农业无人机市占率和行业报告推算

#### B6. 低空运营服务（培训、旅游、巡检、应急等）
- 无人机操控员培训市场快速增长，持证27万+，缺口100万（[千龙网](https://china.qianlong.com/2025/1027/8572802.shtml)）
- 低空旅游、电力巡检、应急救援等运营服务
- 2025年估算：~500-800亿元
- [PROXY — REASONED ESTIMATE]

**Bottom-Up 汇总（2025年）**:

| 细分市场 | 规模估算 | 数据质量 |
|---------|---------|---------|
| 民用无人机（消费+工业） | ~1,450亿元 | 高（多源验证） |
| 无人机物流配送 | ~400亿元 | 中 |
| eVTOL / UAM | ~50-100亿元 | 低（极早期） |
| 低空基础设施 | ~3,000-5,000亿元 | 中（含政府投资） |
| 农业植保 | ~300-500亿元 | 中 |
| 低空运营服务 | ~500-800亿元 | 低 |
| **合计** | **~5,700-8,250亿元** | |

> 取中值约 **~7,000亿元**（核心产业口径）。若加上关联产业（上游材料、芯片、电池等供应链）的乘数效应（约1.5-2x），广义口径可达 **~1.05-1.4万亿元**，与Top-Down的1.5万亿元基本吻合。

### 方法C — 供给侧（Supply-Side）

从已知企业营收推算市场总量：

**头部企业营收（2024-2025年）**:

| 企业 | 估算营收 | 来源 |
|------|---------|------|
| 大疆创新（DJI） | ~500亿元（2024年） | [和讯财经](https://news.hexun.com/2025-08-28/221179178.html) |
| 亿航智能（EHang） | 5.095亿元（2025FY） | [亿航IR](https://ir.ehang.com/news-releases/news-release-details/ehang-reports-fourth-quarter-and-fiscal-year-2025-unaudited/) |
| 中航无人机 | ~30-50亿元 [PROXY] | 上市公司，军民融合 |
| 纵横股份 | ~10-15亿元 [PROXY] | 工业无人机上市公司 |
| 峰飞航空 | 早期阶段 | eVTOL企业 |
| 小鹏汇天 | 早期阶段 | 飞行汽车 |
| 美团无人机 | 早期阶段 | 无人机配送 |
| 丰翼科技（顺丰） | 早期阶段 | 无人机物流 |

**推算逻辑**:
- 大疆在民用无人机市场全球份额超50%（[远瞻慧库](https://www.baogaobox.com/insights/260109000024696.html)），中国市场份额更高
- 头部企业合计营收约550-600亿元
- 大疆占整个低空经济核心产业的比重估算约8-10%（大疆主要在消费级和部分工业级无人机，不含基建和运营服务）
- 推算核心产业规模：550亿 / 8% ≈ 6,875亿元

**Supply-Side 结果**: ~6,000-7,500亿元（核心产业口径，2025年）
