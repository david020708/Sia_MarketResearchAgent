## 2. Founding History — 创始故事与发展历程

### 创始人背景

峰飞航空科技由田瑜（Tian Yu）于2017年在上海创立。田瑜是中国电动航空领域的连续创业者，在无人机和电动飞行器领域拥有超过25年的深耕经验。

田瑜的创业历程可追溯至1990年代末期。他最早从事电动飞行器开发，约2000年前后研制出一款小型电动直升机，在全球范围内引起关注。此后，他联合创办了中翼电子（Zhongyui Electronics），推出了"火鸟"固定翼模型飞机，该产品飞行时间达40分钟，年销售额超过500万元。

田瑜最为人知的早期创业经历是参与创办消费级无人机公司昊翔（Yuneec）。昊翔是电动飞行领域的先驱企业，在田瑜的带领下开发了多款电动马达和单/双座电动飞机。其中双座E430型号获得了FAA认证，并于2009年在美国奥什科什AirVenture航展上公开亮相。2015年，昊翔获得英特尔（Intel）6000万美元投资，成为当时消费级无人机领域的明星企业。

在昊翔进入"休眠状态"后，田瑜于2017年在上海成立峰飞航空科技，将重心转向eVTOL（电动垂直起降飞行器）赛道。2018年，他同步在德国成立AutoFlightX，组建欧洲研发团队，开始双线推进中国和欧洲市场。

## 3. Founding Team and Background — 创始人画像

### 田瑜（Tian Yu）— 创始人、董事长兼CEO

| 项目 | 详情 |
|------|------|
| 教育背景 | 上海交通大学，1991年毕业，无线通信专业 |
| 核心标签 | 连续创业者、电动航空先驱、技术型创始人 |

**职业经历：**

1. **早期电动飞行器开发（1990年代末-2000年代初）**：从事电动飞行器研发，开发小型电动直升机，积累了电动航空领域的核心技术经验。

2. **中翼电子联合创始人**：推出"火鸟"固定翼模型飞机，年销售额超500万元，建立了电动飞行器的商业化能力。

3. **昊翔（Yuneec）联合创始人**：参与创办消费级无人机/电动飞机公司昊翔。昊翔开发了多款电动飞机，其中E430双座电动飞机获FAA认证。2015年获英特尔6000万美元投资。昊翔是全球电动飞行领域的早期先驱之一。

4. **峰飞航空科技创始人（2017年至今）**：创立峰飞航空，全面转向eVTOL赛道，带领公司成为中国eVTOL领域的头部企业和独角兽。

**关于"大疆背景"的说明**：部分媒体报道称eVTOL行业"半壁江山创始人出自同一家企业"，暗示与大疆（DJI）的关联。但根据公开可查资料，田瑜的核心创业经历集中在昊翔（Yuneec）而非大疆。田瑜与大疆之间的直接雇佣关系未获公开资料明确证实。其技术积累主要来自昊翔及更早期的电动飞行器开发经历。

**投资人视角评价**：田瑜是中国电动航空领域经验最丰富的创始人之一，拥有从技术研发到产品商业化的完整闭环经验。上海交大校友网络也为其提供了产业和资本资源——宁德时代创始人曾毓群同为上海交大校友，36氪报道中将CATL投资峰飞描述为"曾毓群投了一位交大学弟"。

## 5. Product Portfolio — 产品组合

峰飞航空拥有业内最完整的eVTOL产品矩阵之一，覆盖货运、载人和大型载客三大场景：

### 5.1 V2000CG "CarryAll"（凯瑞鸥）— 货运eVTOL

| 参数 | 规格 |
|------|------|
| 定位 | 无人货运eVTOL |
| 最大起飞重量 | 2,000 kg |
| 空重 | 1,600 kg |
| 最大载荷 | 400 kg |
| 巡航速度 | 200 km/h |
| 航程 | 250 km |
| 螺旋桨数量 | 13个（10个垂直起降 + 3个推进） |
| 电机数量 | 13台电动机 |
| 机身材料 | 碳纤维复合材料 |
| 机翼构型 | 1个鸭翼 + 1个高鸥翼 + 4个VTOL吊臂 |
| 起落架 | 固定滑橇式 |
| 驾驶方式 | 全自主飞行 |
| 认证状态 | **已集齐CAAC"三证"**（TC/PC/AC），全球首款吨级eVTOL完成全部认证 |
| 商业化状态 | 2025年7月首台交付广州，已进入规模化生产阶段 |

**核心技术亮点**：采用分布式电推进（DEP）设计，通过多螺旋桨冗余提升飞行安全性。垂直起降到前飞巡航的转换过程中无活动舵面或倾转部件，降低了机械复杂度和故障风险。
