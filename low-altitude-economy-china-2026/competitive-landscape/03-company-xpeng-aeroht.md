## 2. Founding History（创始故事）

### 2.1 起源（2013-2020）

赵德力于2013年在广东创立汇天航空航天的前身公司，最初专注于飞行摩托的研发。赵德力本人是"中国飞行摩托第一人"，在38岁时冒着生命危险亲自试飞自研飞行器，展现了极强的冒险精神和技术执着。

### 2.2 与小鹏汽车的结合（2020-2021）

2020年前后，赵德力在42岁时遇到"贵人"——小鹏汽车创始人何小鹏。何小鹏看中了赵德力在飞行器领域的技术积累和创业韧性，决定投资并深度合作。公司正式更名为"小鹏汇天"（XPeng AeroHT），成为小鹏汽车生态体系的核心组成部分。

这一合作的战略逻辑清晰：
- **赵德力**带来飞行器研发经验和创业激情
- **何小鹏/小鹏汽车**带来资金、汽车供应链、智能化技术和品牌影响力
- 双方共同定义了"飞行汽车"这一全新品类——不是纯飞行器，而是陆空一体的交通工具

### 2.3 品牌独立化（2025-2026）

2025年10月，小鹏汇天正式更名为 **Aridge**，发布全新品牌标识。这一更名标志着：
- **品牌国际化**：摆脱对"小鹏"母品牌的依赖，建立独立的全球品牌认知
- **IPO准备**：为港股独立上市铺路，需要清晰的品牌和法人实体边界
- **战略独立**：从小鹏汽车的"子公司"向"关联企业"转型，拥有独立的融资和运营能力

## 5. Product Portfolio（产品组合）

### 5.1 旗舰产品："陆地航母"（Land Aircraft Carrier）

Aridge 的核心产品理念是**分体式飞行汽车**——一辆地面SUV搭载一架可分离的电动飞行器，实现"买车送飞机"的概念。

| 项目 | 详情 |
|------|------|
| 产品名称 | 陆地航母（Land Aircraft Carrier） |
| 产品形态 | 分体式：地面模块（SUV）+ 飞行模块（eVTOL飞行器） |
| 飞行模块 | 电动垂直起降（eVTOL），可从地面模块上自动分离/回收 |
| 地面模块 | 可独立行驶的智能电动SUV |
| 预售价格 | 不超过200万元人民币（约27.5万美元） |
| 国内订单 | 超过5,000台（截至2025年底）；全球预订超7,000台（含中东600台） |
| 量产状态 | 2026年3月批量试产下线；计划2026年下半年正式量产交付 |
| 飞行成本 | 约1万元人民币/小时 |

### 5.2 A868 — 下一代混动飞行汽车

| 项目 | 详情 |
|------|------|
| 发布时间 | 2025年10月（与Aridge品牌更名同步发布） |
| 动力系统 | 混合动力（油电混合），突破纯电续航瓶颈 |
| 航程 | 超过500公里 |
| 座位数 | 6座 |
| 定位 | 长航程飞行出行解决方案，面向城际交通场景 |
| 状态 | 原型机阶段 |

### 5.3 产品路线图逻辑

Aridge 的产品策略分两步走：
1. **近期（2026）**：以"陆地航母"切入市场，主打"陆空一体"的消费级飞行汽车体验，价格锚定200万以内
2. **中期（2027+）**：以A868等混动产品拓展航程和应用场景，从个人消费向城际交通、商务出行延伸

**与竞品的差异化**：亿航EH216-S是纯空中出租车（无地面行驶能力），峰飞Prosperity是城际eVTOL（无地面模块）。Aridge的"陆空一体"路线是独特的产品定位——既是一辆车，又是一架飞机。

| 目标场景 | 个人消费+低空出行 | 城市空中出租车+旅游 | 城际客运 |

### 9.2 Aridge 的差异化定位

Aridge 在中国低空经济赛道中占据了一个**独特的生态位**：

1. **产品差异化**：唯一采用"分体式飞行汽车"路线的头部企业。亿航和峰飞都是纯飞行器，Aridge是"车+飞机"的组合体
2. **客户差异化**：直接面向C端消费者（个人购买），而非B端运营商。这意味着更大的潜在市场规模，但也面临更高的消费者教育成本
3. **价格差异化**：200万元的定价对标高端豪华车（保时捷Cayenne级别），而非航空器价格体系
4. **供应链差异化**：背靠小鹏汽车的汽车级供应链，有望实现比纯航空企业更低的制造成本

### 9.3 竞争风险

- **亿航的先发优势**：亿航已获得全球首张eVTOL型号合格证，已开展商业飞行，在监管认可度上领先
- **峰飞的国际化**：峰飞在欧洲市场的认证和合作布局更为积极
- **新进入者**：吉利旗下沃飞长空、广汽等传统车企也在布局飞行汽车，可能复制Aridge的"车企+飞行器"模式
