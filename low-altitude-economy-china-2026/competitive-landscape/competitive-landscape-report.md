# 中国低空经济竞争格局分析

**报告日期**: 2026年3月
**分析师**: Sia Market Research
**委托视角**: 投资人 / VC — 赛道研究与投资决策

---

## 1. 竞争格局总览

中国低空经济已形成"四极"格局（[搜狐](https://www.sohu.com/a/994167755_120588187)），正从"政策驱动"向"商业落地"跨越。相关企业超16万家（[泰伯网](https://www.taibo.cn/p/99088)），但核心玩家集中在以下几个赛道：

| 赛道 | 竞争格局 | 集中度 |
|------|---------|--------|
| 消费级无人机 | 大疆近乎垄断 | 极高（CR1>70%） |
| 工业级无人机 | 大疆领先，多家上市公司竞争 | 高 |
| eVTOL整机 | 亿航领先，多家创业公司追赶 | 低（早期分散） |
| 无人机物流 | 美团、顺丰、京东三强 | 中 |
| 低空基建（硬件） | 央企+地方国企主导 | 中-高 |
| 低空基建（软件/UTM） | 创业公司+科技巨头 | 低 |
| 通用航空运营 | 中信海直等传统玩家 | 中 |

## 2. 核心企业深度分析

### 2.1 大疆创新（DJI）— 无人机绝对霸主

| 维度 | 数据 |
|------|------|
| 成立 | 2006年，深圳 |
| 上市状态 | 未上市（私有） |
| 2024年营收 | 据报超500亿元（[和讯财经](https://news.hexun.com/2025-08-28/221179178.html)） |
| 净利润率 | 约40%（[和讯财经](https://news.hexun.com/2025-08-28/221179178.html)） |
| 全球市占率 | 消费级>70%，全球无人机份额>50%（[远瞻慧库](https://www.baogaobox.com/insights/260109000024696.html)） |
| 核心产品 | Mavic系列（消费）、Matrice系列（工业）、T系列（农业） |
| 竞争优势 | 技术壁垒、供应链规模、品牌、生态系统 |
| 风险 | 美国制裁压力、消费市场增速放缓 |

**定位**: 低空经济的"基础设施级"企业，类似新能源汽车赛道中的宁德时代。

### 2.2 亿航智能（EHang）— eVTOL先行者

| 维度 | 数据 |
|------|------|
| 成立 | 2014年，广州 |
| 上市状态 | 纳斯达克上市（EH） |
| 2025年营收 | 5.095亿元（创历史新高）（[亿航IR](https://ir.ehang.com/news-releases/news-release-details/ehang-reports-fourth-quarter-and-fiscal-year-2025-unaudited/)） |
| 里程碑 | 全球首张载人eVTOL型号合格证（TC） |
| 核心产品 | EH216-S（载人eVTOL）、EH216-F（消防）、EH216-L（物流） |
| 竞争优势 | 适航认证先发、政府关系、品牌知名度 |
| 风险 | 营收规模仍小、商业化场景有限、竞争加剧 |

**定位**: eVTOL赛道的"开路先锋"，但能否保持领先取决于规模化速度。

### 2.3 小鹏汇天 / Aridge — 飞行汽车新势力

| 维度 | 数据 |
|------|------|
| 成立 | 2013年（原小鹏汇天，2026年更名Aridge） |
| 母公司 | 小鹏汽车（市值约190亿美元） |
| 融资 | Series B累计$2.5亿+，Pre-IPO轮$2亿（[Caproasia](https://www.caproasia.com/2026/03/16/china-19-billion-electric-car-giant-xpeng-plans-flying-car-unit-aridge-previously-xpeng-aeroht-raised-200-million-in-pre-ipo-funding-round-founded-in-2013-as-xpeng-aeroht-by-president-zhao-deli/)） |
| 投资方 | 高瓴GL Ventures、红杉中国（HongShan）（[eVTOL Insights](https://evtolinsights.com/china-xpeng-aeroht-secures-usd250-million-investment-from-series-b-financing-round/)） |
| 核心产品 | 陆空一体飞行汽车 |
| 竞争优势 | 小鹏汽车供应链协同、资本实力、量产能力 |
| 风险 | 适航认证进度、消费者接受度 |

**定位**: "汽车+航空"跨界玩家，差异化路线。

### 2.4 峰飞航空（AutoFlight）

| 维度 | 数据 |
|------|------|
| 成立 | 2017年，上海 |
| 核心产品 | V2000CG（货运eVTOL）、盛世龙（载人eVTOL） |
| 竞争优势 | 技术积累深、多次创造eVTOL飞行纪录 |
| 定位 | 专注eVTOL整机研发制造 |

### 2.5 物流三强

| 企业 | 低空经济布局 | 关键进展 |
|------|------------|---------|
| **美团** | 无人机配送 | 获CAAC全国全域物流运营许可，第四代配送无人机（[Low Altitude Economy Aero](https://lowaltitudeeconomy.aero/evtol-news-and-electric-aircraft-news/cargo-drones/meituan-fourth-generation-drone-caac-national-full-territory-logistics-license)） |
| **顺丰（丰翼科技）** | 无人机物流 | 专注中长距离物流配送 |
| **京东** | 无人机配送 | 农村和偏远地区配送网络 |

### 2.6 上市公司板块

低空经济A股概念股已形成完整板块，已出现7只翻倍股（[新浪财经](https://finance.sina.com.cn/wm/2024-11-26/doc-incxkhuq7567631.shtml)）：

**eVTOL关联**:
- 万丰奥威、中直股份、山河智能、海特高新

**无人机**:
- 航天彩虹、纵横股份、中无人机

**通用航空运营**:
- 中信海直（A股通航龙头）

**低空基建**:
- 莱斯信息（空管系统）、四川九洲（通信导航）

## 5. 竞争趋势判断

### 短期（2026-2027）
- **基建赛道**: 国企/央企主导，创业公司以软件/系统集成切入
- **无人机物流**: 美团、顺丰加速规模化，窗口期收窄
- **eVTOL**: 适航认证竞赛白热化，2-3家企业将获得TC证书

### 中期（2028-2030）
- **行业整合**: 预计大量早期企业被淘汰或并购，CR5将显著提升
- **eVTOL商业化**: 头部企业开始规模化运营，价格战可能出现
- **跨界竞争**: 汽车企业（小鹏、吉利）、科技企业（华为、腾讯）深度介入

### 长期（2030+）
- **平台化竞争**: 类似网约车模式，低空出行平台可能成为最大赢家
- **国际竞争**: 中国eVTOL标准可能成为全球标准，企业出海

## 6. VC投资视角：赛道选择建议

| 赛道 | 投资窗口 | 风险 | 建议 |
|------|---------|------|------|
| eVTOL整机 | 正在关闭 | 高 | 仅投头部2-3家 |
| 无人机物流 | 开放 | 中 | 关注垂直场景（医疗、应急） |
| 低空基建软件/UTM | 开放 | 中-低 | 最佳VC赛道，技术壁垒+政策需求 |
| 低空运营服务 | 开放 | 低 | 轻资产模式，关注区域龙头 |
| 核心零部件 | 开放 | 中 | 电池、飞控、传感器等国产替代 |
| 消费级无人机 | 已关闭 | 极高 | 不建议投资（大疆垄断） |
