## 2. 创立历史与发展历程

### 2.1 化工起源（1928–1970年代）

UCB于1928年在比利时布鲁塞尔由**Emmanuel Janssen**创立，最初定位为多元化化工集团，业务涵盖工业化学品、薄膜（通过子公司Agfa参股）及制药 ([UCB 90年历史回顾](https://www.ucb-usa.com/stories-media/UCB-U-S-News/detail/article/ucbs-90-year-history-then-now-and-future))。Janssen家族自创立伊始便掌握公司控制权，这一格局延续至今，是UCB区别于大多数上市药企的独特治理特征。

### 2.2 向制药转型（1980–2000年代初）

20世纪80至90年代，UCB逐步剥离化工和薄膜资产，将资源集中于制药业务。这一战略转型的关键里程碑是抗过敏药物**Zyrtec（西替利嗪）**的商业成功——Zyrtec在全球范围内成为重磅炸弹级产品，为UCB积累了充足资本以支撑后续的研发转型 ([UCB 90年历史回顾](https://www.ucb-usa.com/stories-media/UCB-U-S-News/detail/article/ucbs-90-year-history-then-now-and-future))。

### 2.3 收购Celltech：进入生物制药（2004年）

2004年，UCB完成对英国生物技术公司**Celltech**的收购，这是公司历史上最具决定性意义的战略举措。Celltech拥有抗TNF生物制剂候选分子CDP870（即后来的certolizumab pegol/赛妥珠单抗），这一收购直接奠定了UCB免疫学生物制剂业务的基础 ([SEC文件，2004年](https://www.sec.gov/Archives/edgar/data/877799/000104746904020291/a2138558zex-99_3.htm))。Celltech同时带来了抗体工程和PEGylation（聚乙二醇化）技术平台，成为后来Cimzia差异化定位的核心技术来源。

### 2.4 Cimzia时代：免疫学领导者（2009–2022年）

- **2008年**：Cimzia（certolizumab pegol）获FDA批准用于克罗恩病
- **2009年**：获批用于类风湿性关节炎（RA），正式进入自身免疫大市场
- **2013年**：扩展适应症至银屑病关节炎（PsA）和中轴型脊柱关节炎（axSpA）
- **2018年**：FDA更新Cimzia说明书，正式纳入妊娠安全数据（基于CRIB研究），成为TNF抑制剂中唯一获得此类标签优势的产品 ([UCB CRIB研究新闻稿](https://www.ucb.com/stories-media/Press-Releases/article/Results-from-CRIB-study-on-CIMZIA-certolizumab-pegol-demonstrate-minimal-to-no-placental-transfer-of-drug-from-mother-to-infant-during-third-trimester-of-pregnancy))

### 2.5 战略升级：收购Ra Pharmaceuticals与新一代管线（2020年至今）

- **2020年**：完成收购**Ra Pharmaceuticals**，获得补体介导疾病领域的创新资产（zilucoplan），拓展免疫学管线边界 ([UCB新闻稿，2020年](https://www.ucb.com/newsroom/press-releases/article/ucb-completes-the-acquisition-of-ra-pharmaceuticals-to-deliver-differentiated-therapies-to-patients))
- **2022年**：收购**Zogenix**，强化神经学（罕见癫痫）管线 ([UCB新闻稿](https://www.ucb-usa.com/stories-media/UCB-U-S-News/detail/article/ucb-acquire-zogenix))
- **2023年**：Bimzelx（bimekizumab）在美国获批用于斑块状银屑病，随后于2024年11月获批PsA、AS、nr-axSpA及HS等多项适应症 ([FDA处方信息，2024年](https://www.accessdata.fda.gov/drugsatfda_docs/label/2024/761151s010lbl.pdf))
- **2025年**：宣布收购**Candid Therapeutics**，引入T细胞衔接器（T-cell engager）技术，进一步丰富免疫学创新管线 ([UCB新闻稿](https://www.ucb.com/newsroom/press-releases/article/ucb-to-acquire-candid-therapeutics-building-upon-its-existing-immunology-pipeline-with-novel-t-cell-engagers))

## 4. RA产品组合

### 4.1 核心产品：Cimzia®（赛妥珠单抗，certolizumab pegol）

**产品概述**

Cimzia是UCB免疫学业务的旗舰产品，也是目前UCB在类风湿性关节炎（RA）领域的核心商业资产。其化学名为certolizumab pegol（赛妥珠单抗），是全球唯一一款经FDA标签认证在妊娠期具有极低胎盘转移风险的TNF抑制剂 ([UCB CRIB研究新闻稿](https://www.ucb.com/stories-media/Press-Releases/article/Results-from-CRIB-study-on-CIMZIA-certolizumab-pegol-demonstrate-minimal-to-no-placental-transfer-of-drug-from-mother-to-infant-during-third-trimester-of-pregnancy))。

| 项目 | 详情 |
|------|------|
| **通用名** | Certolizumab pegol（赛妥珠单抗） |
| **靶点/机制** | PEGylated Fc-free抗TNF-α PEG化Fab'片段；通过选择性结合并中和可溶性和跨膜型TNF-α，阻断其促炎信号，同时因不含Fc区而不触发ADCC/CDC效应 |
| **RA适应症** | 中重度活动性RA（含与甲氨蝶呤联用或单药治疗），尤其适用于有生育需求的女性患者 |
| **其他适应症** | 克罗恩病（CD）、银屑病关节炎（PsA）、强直性脊柱炎（AS）、非放射学中轴型脊柱关节炎（nr-axSpA）、中重度斑块状银屑病（Pso） |
| **给药方式** | 皮下注射；RA维持剂量为200mg每2周或400mg每4周 |
| **美国定价** | 每处方约$3,532（2026年零售价，含2支200mg/vial），年度治疗费用（list price）约$42,000–$85,000+，视给药频率而异 ([DrugPrice.org，2026年](https://www.drugprice.org/drug/cimzia)) |
| **患者援助** | UCB提供CimziaConnect项目，商业保险患者可通过联合付款卡将自付额降至$0–$5/月 ([UCB USA定价信息页](https://www.ucb-usa.com/Responsibility/Affordability/Cimzia-Pricing-Info)) |

**年销售额与市场份额**

- **2024财年**：Cimzia年销售额约为**€10–11亿欧元**区间，处于温和下滑通道，主要受欧洲仿制生物制剂竞争压力影响 ([UCB 2024全年业绩新闻稿](https://www.ucb.com/newsroom/press-releases/article/on-growth-path-for-a-decade-plus-strong-launch-execution-driving-company-growth))。Data unavailable — 精确至百万欧元的产品级数据无法从可访问来源获取
- **2025财年**：Cimzia销售额预计延续下滑趋势，公司整体收入增长完全由Bimzelx驱动 ([Investing.com，2026年2月](https://www.investing.com/news/company-news/ucb-fy-2025-slides-revenue-surges-32-bimzelx-drives-growth-93CH-4532216))
- **RA市场份额**：在TNF抑制剂子类中，Cimzia约占全球RA生物制剂市场3–5%份额（估算），落后于阿达木单抗/Humira及其生物类似药、依那西普及英夫利西单抗，但凭借妊娠安全差异化标签保有稳定细分市场 ([TNF抑制剂全球市场报告，GII Research，2025年](https://www.giiresearch.com/report/tbrc1664773-tumor-necrosis-factor-inhibitor-drugs-global.html))

**妊娠安全差异化优势**

Cimzia最重要的临床差异化特征来自其分子结构：作为Fc-free的PEGylated Fab'片段，Cimzia缺乏其他TNF抑制剂（阿达木单抗、英夫利西单抗、依那西普等）所含的Fc区，因此不经由新生儿Fc受体（FcRn）主动跨胎盘转运。CRIB研究（前瞻性对照研究）证实了妊娠晚期Cimzia从母体到婴儿的胎盘转移量极低至不可测 ([CRIB研究，PMC/NIH](https://pmc.ncbi.nlm.nih.gov/articles/PMC6174965/))。FDA据此于2018年更新其美国说明书，明确纳入妊娠安全数据，这是竞争产品迄今未能复制的监管标签优势 ([UCB Cimzia说明书更新新闻稿](https://www.ucb-usa.com/stories-media/UCB-U-S-News/detail/article/Press-Release-CIMZIA%C2%AE-certolizumab-pegol-label-update-marks-major-advance-for-women-of-childbearing-age-with-chronic-inflammatory-disease-in-the-U-S))。

**2026年最新临床数据**

EULAR 2026大会上，UCB发布了新数据，显示Cimzia在RA患者中无论类风湿因子（RF）水平高低均能提供一致的治疗效果，包括血清阴性患者 ([MarketScreener，EULAR 2026报道](https://ca.marketscreener.com/news/ucb-new-eular-2026-data-show-cimzia-certolizumab-pegol-delivered-consistent-effect-in-rheumatoid-ce7f5ddddd88f325))。这一数据有助于Cimzia在更广泛RA患者人群中巩固其市场地位。

### 4.2 成长型产品：Bimzelx®（比美奇珠单抗，bimekizumab）

Bimzelx是UCB目前增长最快的生物制剂，2025年跨越了**重磅炸弹**（年销售额超10亿美元）门槛，主要驱动力来自化脓性汗腺炎（HS）适应症的强劲放量 ([FiercePharma，2026年2月](https://www.fiercepharma.com/pharma/ucbs-fast-growing-powerhouse-bimzelx-crosses-blockbuster-sales-threshold-back-heavy-hitting); [Spherix Global Insights，2026年3月](https://www.spherixglobalinsights.com/ucbs-fast-growing-bimzelx-leaps-across-blockbuster-sales-threshold-as-hs-momentum-builds/))。

| 项目 | 详情 |
|------|------|
| **通用名** | Bimekizumab-bkzx（比美奇珠单抗） |
| **靶点/机制** | IL-17A和IL-17F双重拮抗剂（humanized IgG1 monoclonal antibody）；同时阻断IL-17A与IL-17F，与仅靶向IL-17A的司库奇尤单抗（secukinumab）和依奇珠单抗（ixekizumab）相比具有更全面的IL-17通路阻断效果 |
| **RA相关适应症** | **目前未获批用于RA**。已批准适应症为：中重度斑块状银屑病（PSO，2023年美国初批）、银屑病关节炎（PsA）、非放射学中轴型脊柱关节炎（nr-axSpA）、强直性脊柱炎（AS）、中重度化脓性汗腺炎（HS）（后四项均于2024年11月获FDA批准） ([FDA处方信息，2024年11月](https://www.accessdata.fda.gov/drugsatfda_docs/label/2024/761151s010lbl.pdf)) |
| **RA管线展望** | 目前无公开的Bimzelx RA适应症III期临床试验数据；UCB的RA战略仍以Cimzia为核心 |
| **2025年销售额** | 超过€10亿（跨越重磅炸弹门槛），精确数字Data unavailable — 无法从可访问来源获取 ([FiercePharma，2026年](https://www.fiercepharma.com/pharma/ucbs-fast-growing-powerhouse-bimzelx-crosses-blockbuster-sales-threshold-back-heavy-hitting)) |
| **EULAR 2026数据** | 3年随访数据显示在PsA和axSpA中持续控制炎症，改善患者报告结局并限制疾病进展 ([UCB EULAR 2026新闻稿](https://www.ucb.com/newsroom/press-releases/article/bimzelxrbimekizumab-data-at-eular-2026-show-early-and-sustained-inflammation-control-can-improve-patient-reported-outcomes-and-limit-disease-progression-in-psa-and-axspa)) |

**对RA市场创业者的战略含义**：Bimzelx虽未直接进入RA适应症，但其在PsA和axSpA（均与RA存在患者重叠）的成功商业化表明UCB具备在免疫学领域快速扩大市场份额的能力，也间接挤压了RA新进入者的可及患者池。

## 7. 技术与知识产权

### 7.1 核心技术平台：PEGylation（聚乙二醇化）

Cimzia的技术基础是**PEGylated Fc-free Fab'片段**技术，其核心特征在于以下三个层面的设计创新：

1. **Fc区缺失**：通过基因工程移除IgG抗体的Fc恒定区，消除Fc介导的效应功能（ADCC/CDC），同时从根本上阻止FcRn介导的胎盘主动转运机制
2. **PEG化修饰**：在Fab'片段上共价连接聚乙二醇（PEG）分子，大幅延长体内半衰期（使皮下注射每2–4周给药一次成为可能），并降低免疫原性
3. **高亲和力TNF结合**：保留了对可溶性和跨膜TNF-α的高亲和力中和活性

PEGylation技术平台最初由**Nektar Therapeutics**开发，并授权给UCB用于certolizumab pegol的开发；FDA批准Cimzia用于克罗恩病时，UCB确认了该技术里程碑 ([Nektar Therapeutics新闻稿](https://ir.nektar.com/news-releases/news-release-details/fda-approval-cimziar-crohns-disease-represents-latest-milestone))。目前该技术IP已完全整合入UCB知识产权体系。

## 8. 竞争定位

### 8.1 UCB在RA市场的竞争优势

**优势一：妊娠安全的不可替代性**

这是UCB在RA市场最坚固的壁垒。Cimzia是唯一一款在FDA说明书中正式记载妊娠期极低胎盘转移风险的TNF抑制剂，且该优势有前瞻性CRIB研究（而非仅凭注册数据库推断）作为证据支撑 ([PMC/NIH妊娠结局研究](https://pmc.ncbi.nlm.nih.gov/articles/PMC6174965/))。在RA患者中，约70–75%为女性，其中育龄期患者占相当比例，这一细分市场是竞争对手无法在近期以标签手段复制的。

**优势二：Fc-free结构的理论安全优势**

缺乏Fc区不仅影响胎盘转运，还理论上降低了Fc介导的脱靶炎症效应风险，这对部分免疫系统脆弱的患者具有潜在临床意义。

**优势三：跨适应症协同商业化能力**

UCB同时销售Cimzia（RA/PsA/AS/CD/Pso）和Bimzelx（PsA/AS/Pso/HS），同一支专科药代团队可服务风湿科和皮肤科，实现规模经济和医生关系深度。

**优势四：长期临床数据积累**

Cimzia自2008年上市至今，已积累超过15年的真实世界安全性与有效性数据，在风湿科医生群体中建立了深厚的信任基础。

### 8.2 UCB在RA市场的竞争劣势

**劣势一：市场份额偏小，缺乏规模领导力**

在TNF抑制剂子类中，Cimzia市场份额显著低于阿达木单抗（即使考虑Humira生物类似药的冲击，AbbVie维持的总市场份额依然远大于Cimzia）、依那西普和英夫利西单抗 ([TNF抑制剂全球市场报告，Grand View Research，2030年](https://www.grandviewresearch.com/industry-analysis/tumor-necrosis-factor-tnf-inhibitor-drugs-market))。

**劣势二：Cimzia生物类似药压力迫近**

欧洲生物类似药（Alvotech/Advanz AVT10）已进入商业化准备阶段，美国市场专利到期后同样面临价格竞争，将持续侵蚀Cimzia的销售额和利润率 ([BioSpace，2025年](https://www.biospace.com/press-releases/alvotech-and-advanz-pharma-enter-into-european-supply-and-commercialization-agreement-for-biosimilar-candidate-to-cimzia-certolizumab-pegol))。

**劣势三：Bimzelx尚未进入RA**

UCB增长最快的资产Bimzelx目前无RA适应症，意味着UCB在RA领域的增长完全依赖一款正在走向成熟期的老产品（Cimzia），缺乏新一代产品接力。

**劣势四：规模与研发资源不及巨头**

与AbbVie、Pfizer、Amgen等大型跨国药企相比，UCB在销售网络覆盖广度、品牌影响力和研发总投入方面均处于相对弱势地位。

欧洲市场已有生物类似药商业化准备进行中（Alvotech/Advanz AVT10），美国市场将紧随其后，价格和销量的双重侵蚀将持续压制Cimzia收入贡献 ([BioSpace，2025年](https://www.biospace.com/press-releases/alvotech-and-advanz-pharma-enter-into-european-supply-and-commercialization-agreement-for-biosimilar-candidate-to-cimzia-certolizumab-pegol))。

**W4. 规模与资源不及跨国巨头**
与AbbVie（Humira/Skyrizi/Rinvoq组合）、Pfizer、Janssen等竞争对手相比，UCB在全球销售网络广度和总研发预算方面存在结构性差距。

### 机会（Opportunities）

**O1. 育龄女性RA患者的未满足需求**
全球RA患者约70–75%为女性，育龄期患者在用药决策中高度重视妊娠安全性。随着医生和患者意识提升，Cimzia的妊娠安全差异化标签仍有进一步渗透率提升空间，特别是在目前仍使用其他TNF抑制剂但有生育需求的患者中。

**O2. 血清阴性RA患者的细分市场**
2026年EULAR新数据证实Cimzia在RF阴性患者中疗效一致 ([MarketScreener EULAR 2026报道](https://ca.marketscreener.com/news/ucb-new-eular-2026-data-show-cimzia-certolizumab-pegol-delivered-consistent-effect-in-rheumatoid-ce7f5ddddd88f325))，这一患者群体历史上被部分医生认为对TNF治疗响应不稳定，新数据打开了进一步渗透的临床论据。

**O3. Bimzelx潜在RA适应症扩展**
基于IL-17A/F在RA滑膜炎中的致病证据，Bimzelx未来存在探索RA适应症的科学基础；若UCB决定推进，将为其在RA市场提供全新的增长动力。

**O4. 新兴市场的渗透空间**
亚洲（中国、印度、韩国、日本等）RA生物制剂市场快速增长，生物制剂渗透率仍低于欧美，UCB有空间通过本地合作伙伴或直接市场准入拓展地理覆盖。

**O5. 下一代免疫学技术（T细胞衔接器）**
Candid Therapeutics收购带来的T细胞衔接器平台代表免疫学治疗的前沿方向，若成功开发，可能为UCB提供超越当前生物制剂的差异化优势 ([UCB新闻稿](https://www.ucb.com/newsroom/press-releases/article/ucb-to-acquire-candid-therapeutics-building-upon-its-existing-immunology-pipeline-with-novel-t-cell-engagers))。

### 威胁（Threats）

**T1. 生物类似药对Cimzia的快速侵蚀（最紧迫威胁）**
欧洲市场生物类似药商业化准备已在进行（2025–2026年），美国市场紧随其后。历史上Humira在欧洲遭遇生物类似药后市场份额快速萎缩的先例提示了风险量级 ([IQVIA欧洲生物类似药竞争影响白皮书，2024年](http://www.iqvia.com/-/media/iqvia/pdfs/library/white-papers/the-impact-of-biosimilar-competition-in-europe-2024.pdf))。

**T2. JAK抑制剂对TNF抑制剂整体类别的冲击**
乌帕替尼（upadacitinib，AbbVie）、巴瑞替尼（baricitinib，Eli Lilly）等口服JAK抑制剂持续扩大市场份额，部分患者和医生因其口服便利性而优先选择JAK抑制剂，压缩所有TNF抑制剂（包括Cimzia）的总体可及患者池。

**T3. IL-6和JAK抑制剂在TNF失效患者中的竞争**
RA二线及以上治疗中，托珠单抗（tocilizumab）、阿巴西普（abatacept）等非TNF生物制剂以及JAK抑制剂提供了有竞争力的替代选择，限制了Cimzia作为二线选项的市场空间。

**T4. 妊娠安全优势的潜在标签追赶**
竞争对手（尤其是Humira生物类似药制造商）存在通过设计专项妊娠安全研究来追赶Cimzia标签优势的可能性，尽管短期内前瞻性数据的生成需要3–5年。

**T5. 付款方/PBM合同压力**
随着生物制剂市场竞争加剧，PBM和保险公司优先使用折扣力度更大的Humira生物类似药，通过处方集地位（formulary tiering）对Cimzia施加价格下行压力。
